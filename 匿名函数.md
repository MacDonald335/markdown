# 匿名函数 与 句柄

# 函数的句柄 ————数据类型
## function_handle
**典型用法是将函数传递给另一个函数**
*类似于指针*
## 语法 
句柄变量 = @(输入参数列表) 表达式
## 创建
```matlab
f = @sin        % f 现在指向 sin 函数
g = @myFunc     % g 指向你自己写的 myFunc 函数
```
把这个指向放在变量里

句柄变量 = @(输入参数列表) 表达式

## 调用
直接用句柄变量名，后面加括号和参数，就像普通函数一样。
``` matlab
y = f(pi/2)     % 相当于 sin(pi/2)，结果是 1
```
 ##  有什么用？
1. 把函数当作参数传给其他函数（比如 integral(@sin, 0, pi)）。

2. 在数据结构中存储函数（比如元胞数组里放不同函数句柄）。

3. 延迟调用：先决定用哪个函数，后面再调用。

4. 代替函数名，实现更灵活的代码。

# 匿名函数
## 解释
**匿名函数是一种没有保存在.m中的，临时的，单条表达式的函数**
**换句话说 匿名函数是创建句柄的一种快捷方式**
## 语法
``` matlab
句柄变量 = @(输入参数列表) 表达式
square = @(x) x.^2;
add = @(a, b) a + b;
hypot = @(x, y) sqrt(x.^2 + y.^2);
```
## 调用方式
和普通函数句柄完全一样：

```matlab
square(5)      % 返回 25
add(3, 4)      % 返回 7
hypot(3, 4)    % 返回 5
```
## 4. 特点
1. 不需要单独写 .m 文件，一行代码搞定。

2. 只能包含一个表达式，不能写 if/for/while 或多条语句（但可用三元表达式 if-else 的替代写法）。

3. 可以捕获外部变量（后面讲）。
