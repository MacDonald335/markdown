# 插值函数充当了预测器
*背景 已经有了一个汇总表记录了不同“空调温度”和“风机转速”下的“机柜温度”。*
## step1 构建：F_temp = scatteredInterpolant(T_air, fan_speed, T_cabinet, 'linear', 'none');
**将两个控制参数（空调温度、风速）作为输入，将结果（机柜温度）作为输出。**
## step2 查询：predicted_temp = F_temp(test_T_air, test_fan);
**用户输入任意的控制参数（例如 T_air=16, fan=7.5），模型直接返回预测结果。**
*目的：根据数据处理  根据已有的实验工况数据 建立一个轻量级的数学公式，用于快速预测未做过实验的工况，或者用于后续的优化控制*
