# obj = @(x) calc_cost(x(1), x(2), load_fixed);
1. 代码结构与语法
@(x)：表示这是一个匿名函数，接受单个输入参数 x
calc_cost(...)：调用预先定义的能耗计算函数
x(1) 和 x(2)：从输入向量 x 中提取的两个关键参数
load_fixed：一个预定义的固定变量，代表服务器负载水平
2. 参数含义与工作原理
## x(1)：代表空调设定温度（单位：℃）
优化过程中会尝试不同温度值（如20℃、22℃、24℃等）
温度越高，空调制冷功耗通常越低，但需确保设备不超温

## x(2)：代表风机转速（单位：百分比）
优化过程中会调整转速值（如60%、75%、90%等）
转速越低，风机功耗越小，但可能影响散热效果

**obj = @(x)调用流程**
1. 初始定义
此时仅定义了目标函数的结构
不会立即执行calc_cost函数
相当于告诉MATLAB："当需要计算目标值时，使用这个方式调用calc_cost"
2. 遗传算法启动时的调用
# 当执行到此处时候
